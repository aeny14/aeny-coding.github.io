A. How Teachable Machine Uses Machine Learning

Using a simple, no-code interface, Teachable Machine uses machine learning to let users train models. In order to train the system how to identify patterns, users input labelled data, such as pictures, sounds, or poses, using a supervised learning approach.

Users create many classes and upload photographs for each category in an image classification project. These photos are processed by teachable machine, which extracts key elements like colours, patterns, and shapes. A model that learns how to differentiate between these classes is then trained. The model can make real-time predictions on fresh data after it has been trained.

This method shows how machine learning can be used to address common issues without requiring advanced programming skills, and it enables users to quickly build intelligent systems.

C. What problem is it intended to solve?

The mask Detection System was created to solve public health and safety issues, particularly during pandemics when mask use is important for preventing spreding of infectious diseases.

It can be difficult and time-consuming to manually monitor mask use in public areas like businesses, schools, and transportation systems. By quickly checking if people follow through to mask requirements, our tool helps in automating the procedure. It can help businesses improve safety and maintain safety procedures.
